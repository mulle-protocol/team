## 빌더 소개

윤성권은 애플리케이션을 직접 개발해 본 백엔드 엔지니어 출신의 인프라 엔지니어입니다. 서비스가 어떻게 만들어지고 어디서 병목이 생기는지를 코드 레벨에서 이해하기 때문에, 개발팀이 실제로 쓰기 좋은 배포 파이프라인과 운영 환경을 잘 설계합니다.

Kubernetes 위에서 컨테이너화된 서비스를 오케스트레이션하고 Istio 서비스 메시로 트래픽·보안·관측성을 다루며 블록체인 노드·RPC 같은 상태 있는(stateful) 인프라까지 안정적으로 운영합니다. 인프라를 세팅하는 데서 멈추지 않고 배포 자동화와 모니터링·알림 체계로 장애를 빠르게 감지하고 복구하는 운영 기반까지 만드는 것을 목표로 합니다.

## 핵심 강점

### 1. 백엔드 개발 경험에 뿌리를 둔 DevOps 역량

- Java / Spring, Python, Node.js / TypeScript로 실제 프로덕션 서버 API와 어드민 시스템을 개발한 백엔드 엔지니어 출신
- 개발과 운영 양쪽을 모두 경험했기에 개발자 경험(DX)을 고려한 CI/CD·배포 환경을 설계
- 애플리케이션 레벨의 성능·병목을 이해한 상태에서 인프라를 튜닝

### 2. Kubernetes · Istio 기반 컨테이너 인프라 운영

- Kubernetes를 이용한 컨테이너 오케스트레이션과 워크로드 운영
- Istio 서비스 메시를 활용한 트래픽 관리, mTLS 보안, 서비스 간 관측성 확보
- Helm / GitOps 기반의 선언적 배포와 형상 관리

### 3. 블록체인 노드 인프라 운영 (Blockchain DevOps)

- Ethereum, BSC, Polygon, Arbitrum, Optimism, Base, Avalanche, Core DAO, Oasys, Solana, Bitcoin, IPFS 등 멀티체인 노드 운영 경험 — Archive / Full 노드 전부 포함
- 노드·밸리데이터·RPC 등 상태 있는 블록체인 인프라의 배포·업그레이드·모니터링·장애 대응
- 팀 서비스가 올라가는 인프라 계층 전반을 책임지는 DevOps 역할

### 4. CI/CD 자동화와 관측성(Observability)

- CI/CD 파이프라인 구축으로 빌드·테스트·배포 자동화
- Prometheus / Grafana 기반의 메트릭·대시보드·알림 체계 구성
- 로그·트레이스·메트릭을 통합해 장애를 빠르게 감지·복구하는 운영 체계 구축

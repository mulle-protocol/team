## 빌더 소개

김기황은 프로토콜과 지갑, 스마트 컨트랙트의 복잡성을 사용자가 실제로 이해하고 이용할 수 있는 제품 경험으로 전환하는 데 집중해 온 엔지니어입니다. React, Next.js, TypeScript를 중심으로 프론트엔드 아키텍처를 설계하고, 지갑 연결, 네트워크 전환, 컨트랙트 인터랙션, 트랜잭션 상태와 오류 처리, 분석 도구 연동, 운영 단계의 성능·신뢰성 개선까지 사용자 접점 전반을 다룹니다.

특히 단순한 화면 구현을 넘어, 금융·온체인 플로우에서 사용자가 어느 단계에 있는지 이해하고 안전하게 다음 행동을 할 수 있도록 만드는 제품 설계에 강점이 있습니다. 이는 GIWA 월렛 내 탑재를 고려한 앱에서도 중요한 역량입니다.

## 핵심 강점

### 1. 프로덕션 Web3 제품을 끝까지 만드는 실행력

- 아이디어와 사용자 플로우를 실제 서비스 수준의 프론트엔드로 구현
- 테스트넷 MVP 이후에도 운영, 성능, 분석, 오류 대응까지 이어지는 제품화 경험
- React, Next.js, TypeScript, Vue 기반의 현대적 웹 애플리케이션 개발

### 2. 복잡한 온체인 플로우를 사용자 경험으로 번역하는 역량

- 지갑 연결, 네트워크 전환, 스마트 컨트랙트 호출, 트랜잭션 상태 처리 구현
- pending, 실패, 재시도 등 불확실성이 큰 블록체인 흐름을 사용자 친화적으로 설계
- DeFi·DEX·BTCFi·크로스체인 제품에서 금융 정보와 온체인 상호작용을 일관된 UI로 제공

### 3. 제품 관점의 프론트엔드 아키텍처

- 기능 출시 속도와 장기 유지보수성을 함께 고려한 구조 설계
- 분석 도구 연동을 통해 사용자 행동을 확인하고 개선할 수 있는 기반 마련
- 여러 이해관계자와 협업하며 제품 요구사항을 구현 가능한 경험으로 구체화

### 4. 멀티체인 환경에서 검증된 빠른 프로토타이핑

- ETHGlobal Taipei에서 Hyperlane 활용 부문 및 멀티체인 USDC 결제 시스템 부문 1위 수상
- HashKey Chain Hackathon에서 Real World PayFi Application Award 1위 수상
- Memecore Memekathon Seoul 2위 수상

해커톤에서의 성과는 아이디어를 제한된 시간 안에 작동하는 제품으로 만들고, 멀티체인·스테이블코인·결제 경험을 사용자 플로우로 완성하는 역량을 보여줍니다.
